# Proactive KYC (pKYC) Dynamic Monitoring Model 🛡️📊

An automated, Python-driven Proof of Concept (PoC) designed to transform traditional periodic Customer Due Diligence (CDD) reviews into an event-driven dynamic monitoring system.

## 🎯 Executive Summary
Traditional periodic KYC reviews (1-3 years) suffer from severe backlogs and consume substantial manpower on low-risk cases. This project processes **10,000 customers** and **50,000 transaction records** to separate low-value cases from high-risk entities requiring immediate manual intervention.

## 🚀 Business Impact (The "Why")
* **90% Resource Optimization:** Safely identifies low-risk and "Ghost" accounts for zero-touch processing.
* **Precision Targeting:** Focuses compliance resources strictly on the **3.5% actual high-risk threats**.
* **CRM Synergy:** Converts mandatory CDD touchpoints into intelligent Sales Leads for Cross-selling.

## 🛠️ Technical Stack & Methodology
* **Language/Tools:** Python (Pandas), HTML, Markdown.
* **Scoring Engine (5-Dimension Alert Framework):**
  1. Risk Score Alert Triage
  2. Transaction Pattern vs. Income
  3. Threshold Evasion (Structuring)
  4. Geographic Anomaly
  5. Alert Synergy (Final Priority)
